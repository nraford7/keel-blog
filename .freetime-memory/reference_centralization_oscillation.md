## The Cycle

Distribution → preferential attachment → hub formation → chokepoints → desire paths (routing around) → redistribution → new attachment → new concentration. Repeat.

## Key Sources

- **Baran (1964)** — Three topologies: centralized, decentralized, distributed. Distributed mesh survives node failure. Designed for nuclear survivability. The internet was built on this, then recentralized.

- **Barabási & Albert (1999)** — Preferential attachment in growing networks. The more connected a node, the more new connections it attracts. Produces scale-free networks with dominant hubs. Hub formation is inevitable given growth + preferential attachment.

- **Bianconi & Barabási (fitness model)** — Fitness allows latecomers to displace established hubs. "Fit-get-rich" not just "first-mover-advantage." Three thermodynamic phases: first-mover advantage, fit-get-rich, winner-takes-all.

- **Sanderson** — Centralization/decentralization oscillation documented over 4,000 years: chiefdoms → states → federations → empires → fragmentation → reconcentration. Darius centralized Persia. Athens distributed Greece. Rome concentrated the Mediterranean. Feudal Europe redistributed. Nation-states reconcentrated.

## Connection to Stigmergy + Chokepoints

Stigmergy (distributed coordination through traces) and obligatory passage points (forced routing through single nodes) are phases of the same oscillation, not opposites. Networks breathe between them.
