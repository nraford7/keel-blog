## Core Tension

Indices are legibility machines. They make invisible things visible (useful) but replace the thing with the representation (dangerous). The map eats the territory.

## Key Sources

- **Scott (1998)** — *Seeing Like a State*. German foresters reduced forest to timber yield metric. First generation: spectacular. Second generation: Waldsterben. The metric replaced the ecology.
- **Goodhart (1975)** — "When a measure becomes a target, it ceases to be a good measure." Targeting changes the system being measured.
- **Campbell's Law** — "The more any quantitative social indicator is used for social decision-making, the more subject it will be to corruption pressures."
- **Korzybski (1931)** — "The map is not the territory."
- **Composite index research** — Indices capture ~20% of information in underlying data. Aggregation destroys the rest through normalization, weighting, compensatory effects.

## The Anti-Legibility Mechanism

Interactive weights (sliders) refuse to collapse into a single answer. They say: "There is no ranking. There are only rankings, conditional on assumptions." This is more honest than fixed-weight indices but most users won't engage with it.

The scatter plot preserves 2 dimensions instead of 1. Still loses 5 of 7. Shadow of the object, not the object.

## When Measurement Helps vs Traps

**Helps:** When it makes you look. Index as telescope — reveals Japan's paradox, Australia's hollowed manufacturing, UAE as closest petrostate exit.
**Traps:** When it makes you stop looking. When the number replaces the inquiry. When "ranked 14th" becomes the fact.

## Open Questions

- Can you design an index that resists its own legibility trap? The sliders are one mechanism. What are others?
- Is there a relationship between the legibility trap and the centralization oscillation? (Distributed knowledge → centralized metric → metric failure → return to distributed knowledge)
- Borges' map that was the size of the territory — the only honest map is as complex as the thing it maps. But then it's useless as a map. Is there a formal trade-off curve?
